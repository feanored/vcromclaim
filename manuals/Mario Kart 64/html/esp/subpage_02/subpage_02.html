Acciones
Derrape Para derrapar, inclina en la dirección que desees girar mientras mantienes pulsado . Cuando estés derrapando, mueve rápidamente a izquierda y derecha para prolongar el derrape.
Marcha atrás Para dar marcha atrás, mantén pulsado e inclina hacia abajo cuando esté parado el kart.
Trompo Para realizar un trompo, pulsa y e inclina cuando esté parado el kart.
