Commencer

Appuyez sur , ou pour passer l'introduction et accéder à l'écran titre. Appuyez à nouveau sur ou pour afficher l'écran de sélection du fichier. Le niveau (Level) indique l'étoile sur laquelle est arrivé Kirby et le pourcentage vous indique où en est votre collection de cristaux. Appuyez sur droite ou gauche sur pour sélectionner un fichier, puis sur haut ou bas pour sélectionner une option.

Start
Sélectionnez Start pour commencer une partie. S'il existe une sauvegarde, la partie reprend où vous l'aviez laissée. Dans ce cas, Kirby aura sa forme normale et vous disposerez de deux vies.

Options
Mini-games (minijeux) Jouez à des minijeux avec jusqu'à trois autres joueurs.
Enemy Info (infos sur les ennemis) Admirez votre collection de cartes d'ennemis que vous avez gagnées dans les jeu de fin.
Theater (cinématiques) Visionnez une séquence déjà vue en jeu.
Settings (paramètres) Vous avez le choix entre du son en Stereo ou en Mono. Vous pouvez également changer l'aspect du panneau de l'écran de jeu.
Note: vous pourrez débloquer une option supplémentaire en progressant dans le jeu.

Delete
Choisissez cette option pour effacer une sauvegarde.
Pour les parties multijoueurs, chaque joueur aura besoin d'une manette compatible avec la Console Virtuelle. Pour plus de détails, vérifiez le mode d'emploi de la Wii.
