Commandes



Remarque : la configuration des commandes présentée ici correspond aux commandes TYPE 1. Elle peut être modifiée dans la rubrique CONTROLS (commandes) du menu principal.
Se déplacer
Se déplacer à droite ou à gauche Appuyez sur gauche sur ou sur pour vous déplacer à gauche et sur droite sur ou sur pour vous déplacer à droite.
Faire une roulade Appuyez deux fois sur droite sur ou sur pour effectuer une roulade à droite. Appuyez deux fois sur gauche sur ou sur pour effectuer une roulade à gauche. Aucun dommage ne vous est infligé lorsque vous réalisez ce mouvement.
Sauter / Faire un double saut Appuyez sur , ou sur pour sauter. Appuyez une seconde fois sur l'un de ces boutons lorsque vous sautez pour faire un double saut et aller encore plus haut.
Attaquer
Changer la façon de marquer une cible Appuyez sur pour passer d'un marquage manuel (vous pointez vous-même la cible) à un marquage automatique (votre arme verrouille automatiquement les ennemis dans la direction visée lorsque vous appuyez sur ou ). Lorsque vous utilisez le marquage automatique, la puissance de vos tirs diminue.
Déplacer le curseur Déplacez le curseur avec ou .
Tirer Appuyez sur pour tirer dans la direction de votre curseur. Maintenez le bouton enfoncé pour tirer en continu.
Attaquer avec une épée Appuyez sur lorsque vous êtes près d'un ennemi pour l'attaquer avec votre épée.
Contre-attaquer Appuyez sur juste avant qu'un missile ou que tout autre projectile ne touche votre personnage pour contre-attaquer dans la direction indiquée par le curseur.

Remarque : vous pouvez configurer le fonctionnement du curseur dans la rubrique CONTROLS (commandes) du menu principal, ou en sélectionnant CONTROL (contrôle) dans le menu Pause. Sélectionnez NORMAL pour que le curseur se déplace dans la direction indiquée quand vous inclinez ou sur . Lorsque vous sélectionnez INVERT (inverser), le curseur se déplace dans la direction opposée à celle indiquée lorsque vous inclinez ou vers le haut ou le bas.
