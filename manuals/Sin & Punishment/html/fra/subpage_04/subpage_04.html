Commencer

A l'écran titre, appuyez sur ou pour accéder au menu principal. Sélectionnez START GAME (commencer une partie) pour commencer à jouer.

Lorsque vous jouez à deux, le premier joueur contrôle les mouvements du personnage et le second ses attaques.

START GAME ou START / CONTINUE (commencer à jouer ou commencer à jouer / continuer) Lorsque vous avez déjà un fichier de sauvegarde, vous pouvez sélectionner l'option START / CONTINUE. Vous pouvez ensuite choisir entre START GAME (commencer depuis le début du jeu) ou CONTINUE (reprendre la partie à partir de votre dernière sauvegarde).
DIFFICULTY (difficulté) Choisissez entre EASY (facile) et NORMAL.
TRAINING (entraînement) Cet entraînement vous permet de vous familiariser avec les commandes de base du jeu.
SOUND (son) Choisissez entre STEREO et MONO.
RANKING (classement) Accédez à l'écran des résultats (RESULTS) pour voir vos meilleures performances selon le score (SCORE), le temps (CLEAR TIME) et le nombre de cibles atteintes (HIT NUMBER).
CONTROLS (commandes) Configurez les commandes et les autres options liées aux commandes de jeu.
SCENE SELECT (choisir un parcours) Regardez le didacticiel, le prologue ou rejouez un niveau.
SAVE DATA (données de sauvegarde) Effacez toutes les données sauvegardées.
BACK (retour) Revenez à l'écran titre.
Pour les parties multijoueurs, chaque joueur aura besoin d'une manette compatible avec la Console Virtuelle. Pour plus de détails, vérifiez le mode d'emploi de la Wii.
