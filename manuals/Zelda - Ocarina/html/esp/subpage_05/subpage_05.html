Acciones
Cambiar la vista Inclina hacia arriba para entrar en el modo de vista en primera persona, luego usa para mirar a tu alrededor. Dentro de algunos edificios inclina para obtener una vista cenital.
Hablar con Navi El icono de Navi aparecerá de vez en cuando en la esquina superior derecha de la pantalla. Cuando lo haga, inclina hacia arriba para hablar con el hada de Link y recibir consejo.
Usar objetos Inclina arriba, abajo, a la izquierda o a la derecha para asignar un objeto a una de las direcciones. Inclina de nuevo en la misma dirección para usar ese objeto.
Tocar la ocarina Inclina en la dirección en la que hayas asignado la ocarina y para tocarla.
Acciones
Mirar adelante Pulsa para colocar la cámara detrás de Link.
Fijar blanco con L Algunos enemigos y objetos se marcarán con un cursor cuando Link se acerque a ellos. Cuando aparezca el cursor, pulsa para fijar la vista en él. Esto se conoce como "fijar el blanco con L" (L TARGETING). Cuando lo realices sobre un objeto o enemigo, usa para moverte sin dejar de tener la vista fija en él. Hay dos modos de fijar el blanco sobre un objeto: SWITCH y HOLD. SWITCH te permite activar y desactivar el cursor pulsando . Con HOLD, el cursor solo aparecerá mientras mantienes pulsado .
