Acciones
Caminar / Correr Inclina ligeramente para caminar o un poco más para correr. Mantén pulsado mientras caminas para fijar el punto de vista de Link hacia delante.
Saltar Link salvará obstáculos, como agujeros o elevaciones del terreno, si lo diriges directamente hacia ellos. Es posible que Link no pueda salvarlos si la distancia es demasiado grande.
Nadar Inclina en la dirección en la que desees dirigirte. Mantén pulsado cuando estés en la superficie para bucear.
Subir escaleras Inclina hacia una escalera para subir. Muévela hacia abajo para bajar.
Acciones
Bajar Pulsa cuando estés colgando de un saliente o una escalera para bajar con seguridad.
Subir Para subirte a algo, inclina y pulsa . En algunas ocasiones, será suficiente.
Empujar / Tirar Pulsa cuando te encuentres cerca de un bloque, una estatua u otro objeto para agarrarlo. Luego, mantén pulsado e inclina para empujarlo o moverlo.
Levantar Pulsa cuando te encuentres cerca de un objeto para cogerlo y levantarlo.
Tirar Cuando quieras lanzar un objeto, usa para empezar a correr y luego pulsa para lanzarlo. Pulsa mientras estás quieto para dejarlo a tus pies.
Lanzar y dejar caer bombas Cuando quieras lanzar una bomba, usa para empezar a correr y luego pulsa para lanzarla. Pulsa mientras estás quieto para dejar la bomba a tus pies.
Abrir objetos Pulsa cuando te encuentres frente a un cofre o una puerta para abrirlos.
Hablar Colócate cerca de una persona y pulsa para hablar con ella. Fija el blanco con L para hablar con gente que esté lejos.
Examinar objetos Colócate cerca de un cartel o un objeto y pulsa para leerlo o investigar.
